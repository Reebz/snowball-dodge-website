# Snowball Dodge - website

Marketing site for the Snowball Dodge iOS game. Static HTML, deployed on Vercel.

- `index.html` - the whole site (self-contained CSS and JS).
- `leaderboard.json` - top scores rendered by the leaderboard section. Seed data for now; replace with real scores.
- `images/` - app icon, character sprites, and gameplay art.

No build step. Vercel serves the files as-is.
